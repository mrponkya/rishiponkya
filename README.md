## 📂 Real Projects

### 🖥️ 01 — Windows Security Monitoring & Threat Detection (Splunk SOC Lab)

> Built a full SOC lab environment using Oracle VirtualBox, Windows Server 2022, and Splunk Enterprise 10.4.0.

**What I did:**
- Configured Windows Event Logging & Sysmon for deep process-level visibility
- Created custom Splunk searches, automated alerts, and live SIEM dashboards
- Simulated real-world security events and performed alert triage
- Investigated suspicious activity and produced formal incident reports
- Monitored critical Windows Event IDs across the environment

**Tools:** `Splunk Enterprise` `Sysmon` `Windows Server 2022` `VirtualBox` `PowerShell`

---

### 🔍 02 — Physical & Virtual Machine Security Audit (BIA Lab)

> Dual-scope cybersecurity audit covering physical security controls and virtual machine exploitation using Kali Linux vs Metasploitable 2.

**What I did:**
- Performed open port enumeration and OS fingerprinting with Nmap
- Identified and exploited weak credentials across multiple services
- Executed controlled exploits and documented findings with risk ratings (Low → Critical)
- Assessed physical security posture and produced a professional audit report with remediation steps

**Tools:** `Kali Linux` `Metasploit` `Metasploitable 2` `Nmap` `Burp Suite`

---

### 📡 03 — Network Exploitation II: Packet Crafting & Analysis

> Advanced network testing covering the full packet lifecycle — from crafting through capture, analysis, and decoding — between Kali Linux and Windows 10 VMs.

**What I did:**
- Captured and analyzed ICMP traffic between VMs using Wireshark
- Ran SYN scans with hping3 targeting RDP (port 3389) and interpreted responses
- Implemented packet crafting pipeline: Assembly → Editing → Replay → Decoding
- Compared Wireshark, tcpdump, Scapy, and hping3 through real test scenarios

**Tools:** `Wireshark` `hping3` `tcpdump` `Scapy` `Kali Linux` `Windows 10`
